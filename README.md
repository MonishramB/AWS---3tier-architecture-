# AWS---3tier-architecture-
AWS 3 Tier Architecture Project using EC2, VPC,Load Balancer
## Service Used 
-VPC 
-Public Subnet 
-Private Gateway
-Internet Gateway
-NAT Gateway
-Application Load Balancer
-EC2 Instances
## Project Descrition
1 created vpc with the subnets public and private
2 configured Internet gateway for internet access
3 deployed application load balancer in public subnet 
4 launched ec2 instances in a private subnet
5 Used Nat gateway for outbound internet access 
6 implemented security groups and routing tables
## purpose 
To build a build a securable, scalable and highly available web application infrastructure in AWS
